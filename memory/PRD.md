# Roti Manis Bakery - Landing Page PRD

## Original Problem Statement
Buatkan landing page untuk toko roti dengan nama "Roti Manis Bakery" lengkap dengan foto-foto produk dengan layout yang profesional, elegant, aesthetic dan terkesan mahal. Fitur wajib: Google Maps lokasi toko. Warna dasar menyesuaikan dengan bisnis toko roti.

## User Personas
- **Local Customers**: Mencari produk roti berkualitas premium
- **Event Planners**: Mencari wedding cake dan kue untuk acara spesial
- **Food Enthusiasts**: Mengapresiasi roti artisan berkualitas

## Core Requirements
- [x] Landing page profesional dan elegant
- [x] Katalog produk dengan foto dan harga
- [x] Integrasi Google Maps untuk lokasi toko
- [x] Form pemesanan online
- [x] Form kontak
- [x] Galeri foto
- [x] Testimoni pelanggan
- [x] WhatsApp button
- [x] Responsive design

## What's Been Implemented (21 Feb 2025)
1. **Hero Section**: Full-screen hero dengan background image dan CTA buttons
2. **About Section**: Informasi tentang bakery dengan stats (15+ tahun, 50K+ pelanggan)
3. **Products Section**: 6 produk dengan harga (Croissant, Sourdough, Wedding Cake, dll)
4. **Gallery Section**: Bento grid galeri dengan 6 foto
5. **Testimonials Section**: 3 testimoni pelanggan dengan rating bintang
6. **Location Section**: Google Maps embed dengan info alamat, jam operasional
7. **Contact Section**: Form kontak elegant dengan WhatsApp integration
8. **Order Modal**: Dialog pemesanan dengan calendar picker
9. **WhatsApp Float Button**: Tombol WhatsApp melayang
10. **Footer**: Links, social media, contact info

## Tech Stack
- Frontend: React + Tailwind CSS + Shadcn UI
- Backend: FastAPI + MongoDB
- Fonts: Playfair Display, Montserrat, Dancing Script
- Colors: Bronze (#8B5E3C), Cream (#F9F5F0), Gold (#D4AF37)
